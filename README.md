🛒 Blinkit Clone (HTML & CSS)

A Blinkit-inspired grocery delivery web UI built using only HTML and CSS. This project focuses on replicating the design and layout of a quick-commerce platform.

---

🚀 Features

- 🏠 Home page with product sections
- 🛍️ Product listing UI
- 🛒 Cart page design (static)
- 🎨 Clean and modern UI
- 📱 Fully responsive design (Mobile + Desktop)

---

🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
